## 二、前端启动（微信小程序）

### 2.1 安装微信开发者工具

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 使用微信扫码登录开发者工具

### 2.2 导入项目

1. 打开微信开发者工具，点击 **「+」** 或 **「导入项目」**
2. 填写项目信息：

| 字段 | 值 |
|------|-----|
| 项目名称 | `provider` |
| 目录 | 选择本项目根目录 `/Users/yanbaohua/Desktop/Y/provider` |
| AppID | `wx911f600d3fc6b0ac` |
| 开发模式 | 小程序 |
| 后端服务 | 微信云开发 |

3. 点击 **「确定」** 导入

### 2.3 开通云开发

1. 在开发者工具顶部菜单栏点击 **「云开发」** 图标
2. 选择环境 `cloud1-d0g8x50g49ba70206`
3. 确保已开通以下服务：
   - 云数据库
   - 云函数
   - 云存储

### 2.4 启动预览

1. 点击工具栏 **「编译」** 按钮
2. 模拟器中即可预览小程序效果
3. 点击 **「预览」** 生成二维码，手机微信扫码真机调试
4. 点击 **「真机调试」** 在手机上实时调试

## 三、后端启动（CloudBase 云函数）

### 3.1 安装 CloudBase CLI

```bash
npm install -g @cloudbase/cli
```

### 3.2 登录 CloudBase

```bash
tcb login
```

浏览器会自动打开，使用腾讯云账号扫码授权。

### 3.3 云函数列表

| 函数名 | 功能 | 超时 | Runtime |
|--------|------|------|---------|
| `initFunctions` | 数据库初始化 + 种子数据 | 30s | Nodejs18.15 |
| `quickstartFunctions` | 快速入门示例 | 15s | Nodejs18.15 |
| `taskFunctions` | 任务创建/领取/审核/结算 | 15s | Nodejs18.15 |
| `userFunctions` | 用户登录/认证/活体/协议签署 | 15s | Nodejs18.15 |

### 3.4 部署云函数

#### 首次部署（全部）

```bash
# 进入项目根目录
cd /Users/yanbaohua/Desktop/Y/provider

# 部署全部云函数
tcb fn deploy --all --force

# 初始化数据库（创建集合+种子数据）
tcb fn invoke initFunctions
```

#### 更新单个云函数

```bash
# 部署指定函数（如 taskFunctions）
tcb fn deploy taskFunctions --force

# 或进入函数目录更新
cd cloudfunctions/taskFunctions
npm install
cd ../..
tcb fn deploy taskFunctions --force
```

#### 使用脚本部署（快速入门函数）

```bash
bash uploadCloudFunction.sh
```

### 3.5 数据库集合

部署 `initFunctions` 后会自动创建以下 11 个集合：

| 集合名 | 说明 |
|--------|------|
| `tasks` | 任务信息 |

| `help_articles` | 帮助文章 |
| `users` | 用户信息 |
| `identity_records` | 实名认证记录 |
| `liveness_records` | 活体认证记录 |
| `agreement_signatures` | 协议签署记录 |
| `task_claims` | 任务领取记录 |
| `settlements` | 结算记录 |
| `tax_records` | 税费记录 |
| `sms_codes` | 短信验证码 |

## 五、常见问题

### Q1: 小程序编译报「云开发环境未找到」

确认项目已关联正确的云开发环境：
1. 开发者工具顶部 → **云开发** 图标
2. 确保当前环境为 `cloud1-d0g8x50g49ba70206`

### Q2: 云函数调用报「Function not found」

云函数尚未部署，执行：
```bash
tcb fn deploy 函数名 --force
```

### Q3: 数据库集合不存在

初始化数据库：
```bash
tcb fn invoke initFunctions
```

### Q4: 真机调试无法调用云函数

1. 确认手机微信已登录小程序开发者账号
2. 检查小程序 AppID 是否正确：`wx911f600d3fc6b0ac`
3. 云函数部署后需等待几秒生效

### Q5: tcb 命令找不到

CloudBase CLI 未安装或未配置环境变量：
```bash
npm install -g @cloudbase/cli
tcb login
```

### Q6: wx-server-sdk 版本问题

云函数依赖 `wx-server-sdk ~2.6.3`，部署前确保安装了正确版本：
```bash
cd cloudfunctions/函数名
npm install
```
